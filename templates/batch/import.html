Required Hosting Fields / 必填托管字段
English Format
  • Site Name - Hosting site (e.g., "Texas DC-01")
  • Client Email - Customer email address
  • Serial Number - Unique miner identifier
  • Miner Model - ASIC model name
  • Hashrate (TH/s) - Actual hashrate
  • Power (W) - Actual power consumption
  • Hosting Fee ($/month) - Optional, defaults to 0
  • Rack Position - Optional (e.g., "A1-R1-U12")
  • IP Address - Optional miner IP
  • Notes - Optional remarks
中文格式
  • 站点名称 - 托管站点（如"Texas DC-01"）
  • 客户邮箱 - 客户电子邮件地址
  • 序列号 - 矿机唯一标识符
  • 矿机型号 - ASIC矿机型号
  • 算力 (TH/s) - 实际算力
  • 功耗 (W) - 实际功耗
  • 托管费 ($/月) - 可选，默认为0
  • 机架位置 - 可选（如"A1-R1-U12"）
  • IP地址 - 可选矿机IP
  • 备注 - 可选说明
Important: Site Name and Client Email must match existing records in the system. Email matching is case-insensitive.
重要提示：站点名称和客户邮箱必须与系统中的现有记录匹配。邮箱匹配不区分大小写。
