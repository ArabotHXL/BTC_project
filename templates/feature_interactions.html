CRM ↔ Hosting Sync

Customer.email → UserAccess.email → UserMiner

CRM客户通过邮箱匹配关联到托管矿机数据，实现跨模块数据同步

Thermal → Miner Control

Temperature Event → Frequency Throttle → CGMiner Command

温控服务检测到高温时自动降频，通过CGMiner API发送控制命令

Curtailment → Miner

Schedule Plan → Miner Selection → Stop/Start Command

限电调度按计划自动停止/启动矿机，支持优先级策略和自动恢复

Real-time Communication / 实时通信

sequenceDiagram
    participant Miner as 矿机 Miner
    participant Edge as 边缘采集器 Edge
    participant Cloud as 云端 Cloud
    participant DB as 数据库 Database
    participant User as 用户 User
    
    Note over Miner,User: 数据采集流程 Data Collection Flow
    
    loop 每60秒 Every 60s
        Cloud->>Edge: 采集请求 Collection Request
        Edge->>Miner: CGMiner API Call
        Miner-->>Edge: 状态数据 Status Data
        Edge-->>Cloud: 遥测上报 Telemetry Report
        Cloud->>DB: 存储数据 Store Data
    end
    
    Note over Miner,User: 温度告警流程 Thermal Alert Flow
    
    Miner->>Edge: 温度数据 Temperature
    Edge->>Cloud: 遥测上报 Telemetry
    Cloud->>Cloud: 检测异常 Detect Anomaly
    alt 温度过高 Temperature High
        Cloud->>Edge: 降频命令 Throttle Command
        Edge->>Miner: 调整频率 Adjust Frequency
        Cloud->>User: 告警通知 Alert
    end
    
    Note over Miner,User: 限电调度流程 Curtailment Flow
    
    Cloud->>Cloud: 检查计划 Check Schedule
    alt 到达限电时间 Curtailment Time
        Cloud->>Edge: 停机命令 Stop Command
        Edge->>Miner: 停止挖矿 Stop Mining
        Cloud->>DB: 记录事件 Log Event
        Cloud->>User: 执行通知 Execution Notice
    end
Polling Model / 轮询模式

CGMiner调度器每60秒轮询所有矿机，采集状态、算力、温度等数据

Event-Driven / 事件驱动

温控保护和限电调度采用事件驱动模式，触发条件时立即执行

Bidirectional / 双向通信

边缘采集器支持双向通信，既上报数据也接收云端控制命令
