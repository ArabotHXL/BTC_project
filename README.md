# BTC Mining Calculator

## 项目描述
一个专业的比特币挖矿收益计算工具，帮助矿场和矿工分析挖矿盈利能力和投资回报率。

## 核心功能
- 实时获取比特币价格、网络难度和哈希率
- 两种独立的挖矿产出计算算法
- 矿场主和客户分开的收益分析
- 电费和BTC价格敏感性分析热力图
- 支持多种矿机型号的参数配置
- 矿场总功率和矿机数量自动换算
- 基于角色的访问控制系统
- 登录记录和IP地理位置跟踪

## 技术栈
- 后端：Flask (Python)
- 数据库：PostgreSQL
- 前端：Bootstrap, Vanilla JavaScript
- 图表：Chart.js
- 地理位置：IP-API

## 计算逻辑
系统使用两种不同的算法计算BTC挖矿产出：
1. **网络哈希率算法**：使用API获取的实际网络哈希率，包含合理性验证
2. **难度算法**：使用网络难度和区块奖励直接计算

## 部署要求
- Python 3.8+
- PostgreSQL数据库
- 互联网连接（用于获取实时数据）
- 环境变量配置（数据库URL等）

## 使用指南
1. 安装依赖：`pip install -r requirements-local.txt`
2. 配置数据库连接（环境变量DATABASE_URL）
3. 运行应用：`gunicorn --bind 0.0.0.0:5000 main:app`
4. 访问网页，登录系统使用计算器

## 角色权限
- **owner**：完全访问权，可查看所有数据和管理用户
- **admin**：可以管理用户访问权限
- **矿场管理**：可以查看矿场主特有数据
- **矿场客人**：只能查看基本计算结果
