## 1. 概述 (Overview)

### 什么是 HashInsight Enterprise

HashInsight Enterprise 是一款专为大型比特币矿场（20MW+）设计的企业级智能托管管理平台。平台整合了实时监控、AI智能诊断、收益计算、CRM客户管理、限电调度等核心功能，帮助矿场运营商和托管客户实现透明、高效、智能的运营管理。

### 目标用户

| 用户类型 | 说明 |
|---------|------|
| **矿场运营商 (Mining Site Owner)** | 拥有或运营大型矿场（20MW+）的企业或个人，负责站点管理、客户服务、限电调度 |
| **托管客户 (Client)** | 在矿场托管矿机的客户，需要监控自己矿机的运行状态和收益 |
| **平台管理员 (Platform Admin)** | 负责平台整体管理、用户权限分配、系统配置 |

### 核心优势

- **透明运营**: 实时矿机数据、收益报表、限电通知，全程透明
- **智能诊断**: AI驱动的告警诊断，快速定位问题根因
- **高效管理**: CRM系统、工单系统、自动化调度，提升运营效率
- **数据安全**: 企业级加密、角色权限控制、审计日志
- **多语言**: 支持中文和英文界面切换

### 功能概览

| 功能模块 | 说明 |
|---------|------|
| 📊 **托管仪表盘** | 站点概览、矿机状态、告警监控 |
| ⚙️ **矿机管理** | 矿机列表、详情查看、批量操作 |
| 🤖 **AI诊断** | 智能告警分析、根因推断、工单生成 |
| ⚡ **限电管理** | 限电计划、手动操作、AI优化建议 |
| 💰 **收益计算器** | 单机计算、批量计算、ROI分析 |
| 👥 **CRM系统** | 客户管理、线索跟进、交易管理 |
| 🎫 **工单系统** | 问题提交、进度跟踪、工单处理 |
| 📈 **报表分析** | 收益报表、运营报表、数据导出 |

## 2. 快速入门 (Quick Start)

### 系统要求

| 要求项 | 最低配置 | 推荐配置 |
|-------|---------|---------|
| **浏览器** | Chrome 80+, Firefox 75+, Safari 13+ | Chrome 最新版 |
| **网络** | 稳定互联网连接 | 10Mbps+ |
| **分辨率** | 1280 x 720 | 1920 x 1080 |
| **JavaScript** | 必须启用 | 必须启用 |

### 如何访问系统

1. 打开浏览器
2. 在地址栏输入平台网址
3. 进入登录页面

```
访问地址: https://calc.hashinsight.net
登录页面: /login
```

### 首次登录步骤

1. **访问登录页面**
   - 在浏览器地址栏输入平台网址
   - 系统自动跳转到登录页面 `/login`

2. **输入登录信息**
   - 在「邮箱」输入框中填写您的注册邮箱
   - 在「密码」输入框中填写您的密码

3. **点击登录**
   - 点击「登录」按钮
   - 等待系统验证

4. **登录成功后**
   - 系统根据您的角色自动跳转到对应页面
   - 平台管理员 → 站点管理页面
   - 矿场运营商 → 我的客户页面
   - 托管客户 → 客户仪表盘

> 💡 **提示**: 首次登录后，建议立即修改初始密码，确保账户安全。

## 3. 用户角色说明 (User Roles)

HashInsight Enterprise 采用基于角色的访问控制 (RBAC) 系统，不同角色拥有不同的功能权限。

### 角色对比表

| 功能 | 平台管理员 (admin/owner) | 矿场运营商 (mining_site_owner) | 托管客户 (client) |
|-----|:----------------------:|:---------------------------:|:----------------:|
| 站点管理 | ✅ 全部站点 | ✅ 自有站点 | ❌ |
| 矿机管理 | ✅ 全部矿机 | ✅ 自有站点矿机 | 👁 仅查看自己的矿机 |
| 客户管理 | ✅ 全部客户 | ✅ 自己创建的客户 | ❌ |
| 用户账户管理 | ✅ | ✅ 为客户创建账户 | ❌ |
| 限电管理 | ✅ | ✅ | 👁 仅查看 |
| 工单系统 | ✅ 处理全部工单 | ✅ 处理站点工单 | ✅ 提交和查看自己的工单 |
| 收益计算器 | ✅ | ✅ | ✅ |
| 报表导出 | ✅ | ✅ | ✅ 自己的数据 |
| CRM系统 | ✅ | ✅ | ❌ |
| AI诊断 | ✅ | ✅ | ❌ |

### 平台管理员 (Platform Admin)

**角色标识**: `admin` / `owner`

**职责说明**:
- 平台整体管理和配置
- 创建和管理矿场运营商账户
- 分配站点给运营商
- 查看全局数据和报表
- 系统配置和安全管理

**登录后默认跳转**: `/admin/site-owners`

**核心权限**:
- 站点管理（创建、编辑、删除）
- 用户管理（创建、权限分配）
- 全局数据查看
- 系统配置

### 矿场运营商 (Mining Site Owner)

**角色标识**: `mining_site_owner`

**职责说明**:
- 管理分配给自己的矿场站点
- 为托管客户提供服务
- 监控矿机运行状态
- 处理告警和工单
- 执行限电调度

**登录后默认跳转**: `/hosting/host/my-customers`

**核心权限**:
- 自有站点的完全管理权限
- 客户管理（创建、编辑、分配）
- 矿机监控和操作
- 工单处理
- 限电管理
- AI诊断功能

### 托管客户 (Client)

**角色标识**: `client`

**职责说明**:
- 查看自己托管矿机的运行状态
- 了解收益情况
- 提交技术支持工单
- 查看限电通知

**登录后默认跳转**: `/hosting/`

**核心权限**:
- 查看自己的矿机列表和状态
- 查看收益报表
- 提交和跟踪工单
- 使用收益计算器
- 查看限电计划

### 4.4 收益计算器 (Calculator)

#### 单机计算器

```
路径: /calculator
```

**输入参数**:

| 参数 | 说明 | 单位 | 示例值 |
|-----|------|------|-------|

**输出结果**:

| 指标 | 说明 |
|-----|------|
| 日收益 (BTC) | 每日挖矿BTC收益 |
| 日收益 (USD) | 每日挖矿美元收益 |
| 日电费 | 每日电力成本 |
| 日净利润 | 日收益减去日电费 |
| 月净利润 | 月度净利润 |
| 年净利润 | 年度净利润 |
| 回本周期 | 预计投资回收时间 |

**操作步骤**:

1. 导航到 `/calculator`
2. 从下拉菜单选择矿机型号
3. 系统自动填充算力和功耗（可手动修改）
4. 输入电费成本
5. 点击「计算」按钮
6. **预期结果**: 页面显示详细的收益计算结果

#### 批量计算器

```
路径: /batch-calculator
```

**功能说明**:
- 同时计算多台矿机的收益
- 支持CSV文件导入
- 汇总统计和对比分析
- 结果导出

**批量计算操作步骤**:

1. 导航到 `/batch-calculator`
2. 下载CSV模板文件
3. 在模板中填写矿机数据
4. 上传填写好的CSV文件
5. 点击「批量计算」按钮
6. **预期结果**: 显示每台矿机的收益和汇总统计

**理解计算结果**:

| 指标 | 计算公式 |
|-----|---------|
| 日收益 | (算力 / 网络算力) × 日出块奖励 × BTC价格 |
| 日电费 | 功耗(kW) × 24小时 × 电费单价 |
| 日净利润 | 日收益 - 日电费 |
| 回本周期 | 矿机价格 / 日净利润 |

### 4.5 AI智能诊断 (AI Features)

#### 告警诊断功能

**入口位置**:
- 事件监控页面 `/hosting/host/event_monitoring`
- 告警卡片上的「AI诊断」按钮

**诊断流程**:

1. 在事件监控页面查看告警列表
2. 找到需要诊断的告警
3. 点击告警卡片上的「AI诊断」按钮
4. 等待AI分析（约3-5秒）
5. 查看诊断结果

**诊断结果内容**:

AI诊断会生成 **Top 3 根因假设**，每个假设包含:

| 内容 | 说明 |
|-----|------|
| 假设描述 | 可能的问题原因 |
| 置信度 | AI对该假设的信心百分比 |
| 证据 | 支持该假设的数据依据 |
| 建议动作 | 推荐的验证或修复步骤 |
| 风险等级 | 问题严重程度评估 |

**诊断示例**:

```
📋 告警类型: 算力下降
🎯 最可能原因: 矿池连接异常 (置信度 75%)

证据:
- 矿池延迟: 320ms (阈值: 200ms)
- 拒绝率: 3.2% (阈值: 2%)

建议动作:
1. 检查网络连接状态
2. 尝试切换备用矿池
3. 重启矿机网络模块
```

#### 工单草稿生成

AI诊断完成后，可以一键生成工单:

1. 查看AI诊断结果
2. 点击「生成工单」按钮
3. AI自动填写工单内容:
   - 问题描述
   - 诊断结果
   - 建议处理方案
4. 确认或编辑工单内容
5. 点击「提交工单」

## 6. 常见操作流程 (Common Workflows)

### 流程1: 创建和管理客户

**完整操作流程**:

1. **在CRM中创建客户**
   - 导航到 `/crm/customers`
   - 点击「新建客户」
   - 填写客户信息 (姓名、邮箱、电话、公司)
   - 点击「保存」

2. **将客户分配到站点**
   - 导航到 `/hosting/host/my-customers`
   - 点击「分配客户」
   - 选择刚创建的客户
   - 选择目标站点
   - 点击「确认」

3. **为客户创建登录账户**
   - 在客户列表中找到该客户
   - 点击「创建账户」
   - 记录系统生成的临时密码
   - 将登录信息发送给客户

4. **验证**
   - 客户使用新账户登录
   - 确认可以查看自己的矿机

### 流程2: 监控矿机健康状态

**日常监控流程**:

1. **登录系统**
   - 使用矿场运营商账户登录

2. **查看仪表盘**
   - 导航到 `/hosting/host/dashboard`
   - 检查关键指标:
     - 在线矿机比例
     - 活跃告警数量
     - 待处理工单

3. **检查告警**
   - 导航到 `/hosting/host/event_monitoring`
   - 查看告警列表
   - 按严重程度排序

4. **处理异常**
   - 对于Critical告警，立即使用AI诊断
   - 对于Warning告警，记录并安排处理
   - 对于Info通知，了解即可

5. **查看矿机详情**
   - 点击告警中的矿机链接
   - 查看详细运行数据
   - 查看历史趋势图

### 流程3: 使用AI诊断响应告警

**AI诊断操作流程**:

1. **发现告警**
   - 在事件监控页面看到新告警
   - 告警类型: 矿机离线 / 算力下降 / 温度过高

2. **启动AI诊断**
   - 点击告警卡片上的「AI诊断」按钮
   - 等待分析完成 (3-5秒)

3. **查看诊断结果**
   - 阅读Top 3根因假设
   - 查看每个假设的置信度
   - 理解证据数据

4. **执行建议动作**
   - 根据建议动作进行验证
   - 如需现场处理，生成工单

5. **创建工单**
   - 点击「生成工单」
   - 确认AI生成的工单内容
   - 补充必要信息
   - 提交工单

6. **跟踪处理**
   - 在工单系统跟踪处理进度
   - 确认问题解决后关闭工单

### 流程4: 计算挖矿收益

**收益计算流程**:

1. **选择计算方式**
   - 单机计算: `/calculator`
   - 批量计算: `/batch-calculator`

2. **输入参数** (单机计算)
   - 选择矿机型号
   - 确认算力和功耗数据
   - 输入电费单价

3. **查看结果**
   - 日/月/年收益预测
   - 电费成本估算
   - 净利润计算
   - 回本周期预测

4. **批量计算** (可选)
   - 下载CSV模板
   - 填写多台矿机数据
   - 上传文件
   - 查看汇总结果

5. **导出报告**
   - 点击「导出」按钮
   - 选择格式 (PDF/Excel)
   - 下载保存

### 流程5: 创建技术支持工单

**工单提交流程**:

1. **进入工单系统**
   - 导航到 `/hosting/host/tickets` (运营商)
   - 或 `/hosting/client/tickets` (客户)

2. **创建新工单**
   - 点击「新建工单」按钮

3. **填写工单信息**
   - 选择工单类型:
     - 硬件问题
     - 网络问题
     - 软件问题
     - 账单咨询
     - 其他
   - 填写标题 (简述问题)
   - 填写详细描述
   - 选择优先级
   - 关联相关矿机 (可选)

4. **提交工单**
   - 确认信息无误
   - 点击「提交」按钮

5. **跟踪进度**
   - 在工单列表查看状态
   - 查看运维人员回复
   - 补充信息或回复评论

6. **确认解决**
   - 问题解决后确认工单
   - 工单状态变为「已解决」

## 7. 故障排除 (Troubleshooting)

### 常见问题与解决方案

#### Q1: 无法登录系统

**可能原因**:
- 用户名或密码错误
- 账户被锁定
- 账户权限未开通

**解决方案**:
1. 确认邮箱地址输入正确
2. 检查密码是否正确（注意大小写）
3. 使用「忘记密码」重置密码
4. 如多次失败，账户可能被锁定，等待30分钟后重试
5. 联系管理员确认账户状态

#### Q2: 页面显示「访问被拒绝」

**可能原因**:
- 当前角色无权访问该功能
- 会话已过期

**解决方案**:
1. 确认您的账户角色是否有权限访问该页面
2. 尝试重新登录
3. 联系管理员确认权限设置

#### Q3: 矿机数据不更新

**可能原因**:
- 矿机离线
- 采集器连接异常
- 网络延迟

**解决方案**:
1. 检查矿机网络连接
2. 在矿机管理页面查看「最后更新时间」
3. 联系现场检查矿机状态
4. 提交工单让运维人员排查

#### Q4: AI诊断返回错误

**可能原因**:
- 系统繁忙
- 数据不足无法分析
- 服务暂时不可用

**解决方案**:
1. 等待几秒后重试
2. 刷新页面后再次尝试
3. 如持续失败，提交工单反馈

#### Q5: 计算器结果不准确

**可能原因**:
- BTC价格/难度数据延迟
- 输入参数有误

**解决方案**:
1. 确认输入的算力和功耗数据正确
2. 查看页面显示的BTC价格和难度是否最新
3. 等待几分钟让数据更新后重新计算

#### Q6: 无法创建客户账户

**可能原因**:
- 客户邮箱已被使用
- 缺少必要信息

**解决方案**:
1. 确认客户邮箱未被其他账户使用
2. 确认已填写所有必填字段
3. 刷新页面后重试

#### Q7: 限电计划未执行

**可能原因**:
- 计划时间设置错误
- 站点/矿机选择有误
- 系统调度延迟

**解决方案**:
1. 检查计划的开始时间是否正确
2. 确认选择的站点和矿机正确
3. 检查计划状态是否为「待执行」
4. 联系运维人员排查
