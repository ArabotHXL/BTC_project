# 系统流程图文档

本文件夹包含了BTC挖矿计算器系统的三个主要流程图：

## 1. 系统架构流程图 (system_architecture_flow.svg)
展示了系统各模块之间的关系和层次结构，包括：
- 用户界面层
- 应用逻辑层 
- 数据访问层
- 外部服务层
- 用户角色和权限

## 2. 系统流程图 (system_process_flow.svg)
展示了用户交互和主要功能流程，包括：
- 用户认证流程
- 挖矿计算器流程
- 电力管理流程
- CRM系统流程

## 3. 数据流图 (system_data_flow.svg)
展示了数据库实体关系和数据流向，包括：
- 核心数据实体与关系
- 主要数据表结构
- 外部数据流

这些流程图提供了系统的全面视图，从不同角度展示了各组件间的交互。
