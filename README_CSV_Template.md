# 批量矿机计算器 CSV 模板说明

## 支持的格式

### 完整格式（推荐）
```csv
Model,Quantity,Power,Price,Electricity,Hashrate,DecayRate
```

### 简化格式（无衰减率）
```csv
Model,Quantity,Power,Price,Electricity
```

## 字段说明

| 字段名 | 说明 | 单位 | 示例 | 必填 |
|--------|------|------|------|------|
| Model | 矿机型号 | - | Antminer S19 Pro | ✓ |
| Quantity | 数量 | 台 | 10 | ✓ |
| Power | 功耗 | 瓦特(W) | 3250 | ✓ |
| Price | 价格 | 美元($) | 2500 | ✓ |
| Electricity | 电费 | 美元/千瓦时 | 0.08 | ✓ |
| Hashrate | 自定义算力 | TH/s | 110 | 可选 |
| DecayRate | 算力衰减率 | %/月 | 0.5 | 可选 |

## 重要说明

1. **矿机型号**: 支持以下预设型号
   - Antminer S19, S19 Pro, S19j Pro, S19 XP
   - Antminer S21, S21 Pro, S21 XP, S21 Hyd, S21 Pro Hyd, S21 XP Hyd
   - Antminer T19
   - WhatsMiner M50, M50S, M53S, M56S
   - AvalonMiner 1366, 1466

2. **算力衰减率**: 
   - 范围：0-5% 每月
   - 用于长期ROI预测
   - 如果不填写，默认为0

3. **自定义算力**:
   - 如果填写，将覆盖型号默认算力
   - 用于测试不同算力场景
   - 如果不填写，使用型号默认值

4. **文件格式**:
   - 必须是 CSV 格式
   - 第一行必须是字段名（表头）
   - 数据从第二行开始

## 使用步骤

1. 下载模板文件：`batch_calculator_template.csv`
2. 根据您的矿机配置修改数据
3. 保存为 CSV 格式
4. 在批量计算器中上传文件

## 常见问题

**Q: 为什么上传后算力显示为0？**
A: 请检查CSV格式，确保字段名正确，数据格式无误。

**Q: 可以混合使用不同的矿机型号吗？**
A: 可以，系统会自动识别每种型号的默认参数。

**Q: 算力衰减率如何计算？**
A: 使用复合衰减公式：有效算力 = 初始算力 × (1 - 衰减率/100/30)^天数

**Q: 最多可以上传多少台矿机？**
A: Pro用户支持最多999,999台矿机的批量计算。
