# Analytics Dashboard 相关文件清单
# Generated on 2025-08-22

## 核心后端文件
1. app.py - 主应用文件，包含 /analytics_dashboard 路由处理

3. advanced_algorithm_engine.py - 高级算法引擎 (Phase 3, 10模块)
4. language_engine.py - 增强版多语言引擎

## 技术特性
- Phase 3 高级算法引擎 (10个模块: A-J)
- 实时Bitcoin价格和网络数据
- 中英文双语界面
- 响应式设计 (移动端友好)
- 机构级订单执行优化
- TWAP执行和滑点控制
- 实时技术指标计算
- 专业量化交易信号

## 数据流
用户访问 /analytics_dashboard → 
app.py 路由处理 → 
analytics_engine.py 获取实时数据 → 
advanced_algorithm_engine.py 生成信号 → 
templates/analytics_dashboard_new.html 渲染 → 
static/js/main.js 实时更新数据

## 核心功能
1. 实时市场数据监控 (BTC价格、算力、难度)
2. 高级交易算法信号 (10模块系统)
3. 资金管理和投资组合分析
4. 技术指标可视化
5. 多语言支持
6. 订单执行优化建议
