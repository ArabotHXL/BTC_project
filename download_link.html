🔗 Deribit Analysis Package

📦 包内容

  • deribit_web_routes.py - Flask路由和API端点
  • multi_exchange_collector.py - 多交易所数据聚合器
  • deribit_options_poc.py - 核心数据采集引擎
  • deribit_analysis.html - 响应式Web界面
  • deribit_trades.db - 历史交易数据库 (443KB)
  • requirements.txt - Python依赖列表
  • README.md - 详细文档说明
  • INSTALL.md - 安装部署指南
📥 下载完整包 (tar.gz)

🎯 核心功能

  • 多交易所集成 (Deribit + OKX + Binance)
  • 实时BTC期权价格分析 ($118,475+)
  • 交易方向分布可视化
  • 价格区间智能分桶
  • API状态实时监控
  • 历史数据持久化存储
  • 响应式中英文界面

🚀 快速开始

# 1. 解压文件
tar -xzf deribit_analysis_package.tar.gz

# 2. 安装依赖
pip install -r requirements.txt

# 3. 集成到Flask应用
from deribit_web_routes import deribit_bp
app.register_blueprint(deribit_bp)

# 4. 访问页面
http://localhost:5000/deribit-analysis
