# 智能电力削减管理系统使用指南

## 系统概述

智能电力削减管理系统是一个基于矿机健康状态和效率自动管理电力削减策略的工具。系统可以根据矿机的实际运行状况，智能地选择哪些矿机应该关停以实现电力削减目标，同时最大化保持总体挖矿效率。

## 核心功能

1. **矿机健康评分**：基于算力、错误率、温度等指标自动评估矿机健康状况
2. **智能分级**：将矿机分为A/B/C/D四个等级，便于管理
3. **优先关停策略**：优先关停状态较差和效率较低的矿机
4. **轮换计划**：自动生成矿机轮换计划，延长矿机寿命
5. **实时监控**：直观的仪表盘展示系统状态和矿机数据

## 快速开始

### 1. 运行系统

```bash
python run_power_management.py
```

系统将自动初始化1000台虚拟矿机数据（实际应用中应连接到真实矿机），并启动Web界面。

### 2. 访问管理界面

打开浏览器访问：http://localhost:5000/

### 3. 应用电力削减策略

1. 使用左侧控制面板中的滑块调整电力削减目标百分比
2. 点击"应用电力削减策略"按钮执行
3. 系统将自动选择需要关停的矿机，优先关停D级和低效率C级矿机

### 4. 生成矿机轮换计划

1. 点击"生成矿机轮换计划"按钮
2. 系统会显示轮换计划，包括计划关停和计划启动的矿机
3. 点击"执行轮换计划"按钮实施轮换

## 系统配置

### 矿机健康评分算法

系统使用综合算法计算矿机健康得分(0-100分)，考虑因素包括：

- 算力表现 (0-40分)：实际算力与标称算力的比例
- 错误率 (0-30分)：错误率每增加1%扣6分
- 温度 (0-20分)：温度超过80°C开始扣分
- 风扇状态 (0-10分)：风扇速度异常将扣分

### 矿机分级标准

- **A级 (90-100分)**：状态优秀
- **B级 (70-89分)**：状态良好
- **C级 (50-69分)**：状态一般
- **D级 (0-49分)**：状态较差

### 优先关停策略

1. 首先关停所有D类矿机
2. 如果还需要更多削减，按效率从低到高关停C类矿机
3. 如果仍不够，按效率从低到高关停B类矿机
4. 如果还不够，按效率从低到高关停A类矿机

## 实际部署说明

在实际矿场环境部署时，需要修改`smart_power_manager.py`中的`MinerMonitor`类，实现与实际矿机API的对接。主要需修改以下函数：

1. `collect_miner_data`：从实际矿机API采集数据
2. `MinerController.shutdown_miner`：实现真实矿机关机命令
3. `MinerController.start_miner`：实现真实矿机开机命令

## 注意事项

1. 本系统目前使用模拟数据进行演示，实际应用需连接到真实矿机
2. 确保矿机API可访问并正确配置IP地址
3. 首次运行时会创建几个缓存文件用于保存系统状态
